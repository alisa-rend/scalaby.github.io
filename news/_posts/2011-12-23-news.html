Новое

Вышел Specs2 version 1.7! – Specs2 – фреймворк для написания исполняемых спецификаций. Больше информации о версии 1.7 можно найти на ресурсе implicit.ly

Вышел sbt-protobuf v0.2.2! sbt-protobuf – плагин для sbt, генерирующий по файлам с описание протоколов *.proto соответсвующие java-файлы (вызывая protoc).

Miles Sabin (@milessabin) выпустил shapeless 1.0. Shapeless рассматривает обобщенное (политиповое) программированиние на Scala. Это очень небольшая и интересная библиотека. Miles описал проект в блоге.

Вышел sbteclipse 2.0.0-M1. sbteclipse – плагин для sbt для создания создания проектов для Eclipse. Версия 2.0.0 полностью переписана (по сравнению с веткой 1.x).

Также вышел sbt-cloudbees-plugin 0.4.0." Этот плагин для SBT позволяет деплоить приложения в облако CloudBees RUN@Cloud PaaS. Версия 0.4.0 была полностью переписана для SBT 0.10+.

Scala Micro-Benchmarking Template, обертка вокруг гугловской библиотеки caliper.

Наконец, Scala IDE for Eclipse 2.0 Final. Массивное обновление включает новый сайт с документацией и скринкастам наравне с многочисленными улучшениями и исправлениями.
