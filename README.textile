h1. This is the data for "scala.by":http://scala.by site

It is automatically transformed by "Jekyll":http://github.com/mojombo/jekyll into a static site whenever we push this repository to GitHub.

h1. Test locally

* Install Ruby
* Install Bundler `gem install bundler`
* Run `bundle install` once
* Run `bundle exec jekyll serve`
* Local version is running at http://localhost:4000

Detailed instructions are located "here":https://help.github.com/articles/using-jekyll-with-pages/
